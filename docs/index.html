🔍 Desktop Translator

快捷键框选屏幕任意区域 → OCR 识别 → 自动翻译 → 浮窗显示

Windows · Python 3.10+ · Apache 2.0

快速开始

  1. 安装 Python 3.10+
  2. 安装 Tesseract OCR（勾选 English 语言包）
  3. pip install -r requirements.txt
  4. python desktop_translator_cn.py
  5. 按 Ctrl + Shift + S 框选翻译

版本选择

文件翻译源适用场景
desktop_translator.py Google 翻译 海外 / 已开代理
desktop_translator_cn.py 百度 → 有道 → 必应 🏠 国内直连，无需代理

两个版本除翻译源外完全相同，任选其一即可。

命令行安装

# 克隆仓库
git clone https://github.com/l2026/desktop-translator.git
cd desktop-translator

# 安装依赖
pip install -r requirements.txt

# 运行国内版 (推荐)
python desktop_translator_cn.py

# 运行海外版
python desktop_translator.py

# 自测
python desktop_translator.py --test

打包为 exe

pip install pyinstaller

# 国内版
pyinstaller --noconsole --onefile --collect-all spellchecker desktop_translator_cn.py

# 海外版
pyinstaller --noconsole --onefile --collect-all spellchecker desktop_translator.py

# 生成的 exe 在 dist/ 目录
# 需要将 Tesseract 安装目录下的 tesseract.exe 和 tessdata/ 复制到 exe 同级
